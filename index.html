LinkUp A Advanced Chatroom Application

The LinkUp A Advanced Chatroom Application is a high-level project built using Python with advanced authentication, secure communication, and modern backend frameworks like Flask. It incorporates features such as chatroom management, synchronized chatting, chat history, and file sharing. ...

Authentication: The application employs a robust authentication system where users can create accounts and log in securely. Passwords are stored in a bcrypt hash format to enhance security. For an added layer of protection, asymmetric and symmetric keys are utilized for secure communication and data exchange. Proper error messages are displayed for failed authentication attempts.

Chatroom Management: Users can create new chatrooms, join existing ones, and share chatroom names with others. This ensures seamless collaboration and engagement.

Chatroom Search: Users can search their joined chatrooms to continue conversations conveniently.

Synchronized Chat: Synchronized real-time chat is implemented using Socket.IO, ensuring that messages are instantly visible to all logged-in users in the chatroom.

Chat History: Chats are persistently saved in the database, allowing users to view previous messages when they log in again or join a chatroom for the first time.

Delete Chat: Users can delete recently sent messages, while older messages are protected from deletion to maintain conversation integrity.

File Sharing: The platform supports file sharing, enabling users to send files up to 32 MB in size and up to 8 files in a single row for better collaboration.

Audio Messaging: Audio messaging is included for enhanced communication, providing a convenient and expressive way to interact.

This extreme-level project combines advanced technologies, modern backend frameworks like Flask, and secure practices to deliver a scalable and highly functional chatroom application that meets diverse communication needs.

Av-Evasion

The Av-Evasion project is a cyber-security related project that generates custom payload that evades anti-version detection methods.

This custom project tries to evade anti-virus detection via various means.

Evading Static Analysis: In static analysis phase an anti-virus tries to match payload's signature to a malware signature database. But this script encrypts the original payload thus avoids the static analysis.

Evading Dynamic Analysis: In dynamic analysis phase an anti-virus tries to execute the malware code and tries to detect malicious event. To avoid dynamic analysis the script uses various means:
1. Use random sleep time in the begining of the executable in hopes to time out the dynamic analysis process.
2. Use native windows application to inject payload. This script injects payload to windows native application like notepad.exe/calc.exe in hopes that the memory will not be scanned by av as it's a trusted source.
3. Use xor encrypted payload. The payload will be decrypted during runtime thus avoiding av dynamic analysis


Link:

https://github.com/alpha-hexor/av-evasion/
